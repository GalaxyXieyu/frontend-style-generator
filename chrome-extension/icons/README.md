# 图标文件

## 需要的图标尺寸

- `icon16.png` - 16x16 像素（工具栏小图标）
- `icon48.png` - 48x48 像素（扩展管理页面）
- `icon128.png` - 128x128 像素（Chrome Web Store）

## 快速生成图标

### 方法 1: 使用 Emoji
访问 https://favicon.io/emoji-favicons/artist-palette/
下载调色板 emoji 图标

### 方法 2: 在线生成
访问 https://www.favicon-generator.org/
上传图片或使用文字生成

### 方法 3: 使用 Figma/Sketch
设计自定义图标，导出 3 个尺寸

## 临时方案

如果暂时没有图标，插件仍可正常运行，Chrome 会显示默认图标。
